## Comandos Interessantes para saber (basicos) 0-0

 Atualizar tudo:

```bash
sudo pacman -Syu
```


 Instalar um programa:

```bash
sudo pacman -S nome-do-pacote
```



 Procurar pacote:

```bash
pacman -Ss termo
```



 Remover um pacote:

```bash
sudo pacman -Rs nome-do-pacote
```
